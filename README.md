# today-duty
## For who?
교대 근무자를 위한 앱입니다. 매일 변동되는 업무 스케줄을 쉽게 기록하고 확인할 수 있습니다.

## Why today-duty?
실제로 교대 근무를 하는 지인이 사용하는 앱의 불편사항을 듣고 만들어볼까? 해서 만들게 되었습니다. 필수적인 기능만 구현할 예정입니다.

## Tech-Stack
- React-Native  
- Expo CLI
